## Уровни авторитета
1. Canon (`docs/source/*`): нормативные требования.
2. Checklists (`docs/source/checklists/*`): программа выполнения и верификации.
3. Working docs (`docs/*.md`, `docs/*/*.md`): объяснение и операции.
4. Portal docs (`docs/portal/*`): навигация и UX документации.

## Запреты
- Нельзя фиксировать новые нормативные контракты в рабочих документах.
- Нельзя противоречить канону в overview/how-to.
- Нельзя закрывать checklist-пункты без проверок и артефактов.
