# Навигация Портала

## Source of truth (обязательно)
- `docs/README.md`
- `docs/source/README.md`
- `docs/source/checklists/CHECKLIST_00_MASTER_ART_REGART.md`

## Правила
1. Портал не меняет смысл канона, а только связывает материалы.
2. Каждый portal-документ обязан содержать блок `Source of truth`.
3. Ссылки на REGART обязательны для интеграционных страниц.
4. Для новых страниц в портале создаются RU+EN версии одновременно.

## Якоря и ссылки
- Использовать относительные пути внутри `docs/`.
- Не использовать локальные пути (`/home/...`) в markdown.
- Для внешних ссылок фиксировать полные `https://...`.
