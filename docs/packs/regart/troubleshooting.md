# REGART troubleshooting

Частые кейсы:
- `ui.graph.empty` — пустой граф, проверка источников и bridge.
- `upstream_error` — ошибка upstream, проверка connectivity и retries.
- `bridge_backlog_recovered` — bridge восстановил доставку после локального backlog.

При несовместимости версии pack/core см. runbook `pack_incompatible`.
