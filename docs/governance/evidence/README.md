# Evidence Ledger

## Source of truth
- `docs/governance/evidence/evidence_ledger.yaml`
- `docs/source/checklists/CHECKLIST_00_MASTER_ART_REGART.md`
- `scripts/ci/check_evidence_ledger.sh`

## Назначение
Этот каталог хранит фактические артефакты выполнения этапов (логи, скриншоты, отчёты), чтобы прогресс проекта был проверяемым и наглядным для внутренних и внешних наблюдателей.

## Правило
Если этап в MASTER закрыт (`[x]`), для него обязана существовать запись в `evidence_ledger.yaml` и ссылки на реальные файлы/команды проверки.

## Клиентская витрина
Публичное представление прогресса и доказательств: `docs/portal/DELIVERY_EVIDENCE.md`.
