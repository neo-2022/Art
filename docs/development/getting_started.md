# Getting started (RU)

## Rust
- `cargo fmt --check`
- `cargo clippy`
- `cargo test`
- `CORE_CONFIG_PATH=config/core.toml cargo run -p art-core`

## Browser/Node
- `npm ci`
- `npm run lint`
- `npm run test`
- `npm run build`

## Security smoke
- `gitleaks detect --no-git`
- `npx license-checker --production --summary`
