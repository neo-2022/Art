# Documentation Authority Model

## Source of truth
- `docs/portal/DOC_AUTHORITY.md`
- `docs/README.md`
- `docs/source/README.md`

## Levels
1. Canon (`docs/source/*`) is normative.
2. Checklists (`docs/source/checklists/*`) are execution law.
3. Working docs (`docs/*.md`, `docs/portal/*`, `docs/rag/*`) must not contradict canon.

## Change rule
If canon changes, dependent portal/how-to pages must be updated in the same change set.
