## Core Terms
- `Incident`: a managed operational failure context.
- `DNA`: deterministic signature of a problem class.
- `Evidence`: verifiable data reference supporting claims.
- `Claim`: hypothesis/decision/explanation linked to evidence.
- `Gap Event`: `observability_gap.*` event showing observability/control gap.
- `InvestigationDoc`: reproducible investigation artifact.
